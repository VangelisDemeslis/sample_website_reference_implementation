Important HTML tags

These are some important HTML tags:

Tag Name Purpose
a anchor make a link
em emphasis emphasize text (usually rendered as italics)
strong make text strong strengthen text (usally rendered as boldfacd)
ol ordered list list elements in numerical order
ul unordered list list elements whose order doesn't matter
img image insert an image on the current page
