Important HTML tags

This table doesn't include all HTML tags, or even most of them, but it does list some of the most important ones.

Tag Name Purpose
a anchor make hyperlink
em emphasis emphasize text (usually rendered as italics)
strong strong make text stronger (usually rendered as bold)
ol ordered list list elements in numerical order
ul unordered list list elements whose order doesn't matter
li list item include an item in either an ordered or unordered list
img image include an image
table table include a table
tr table row include a row of data
th table heading make a table heading
td table data include a table data cell
